# Aura eCommerce
Uma interface de e-commerce moderna e responsiva construída com **Bootstrap 5**.[1] O foco deste projeto é fornecer uma estrutura sólida de frontend para lojas virtuais, com navegação entre múltiplas páginas e gerenciamento de estado via *Vanilla JavaScript*.
## Páginas do Projeto

- **Home:** Banner principal e destaques.
- **Loja:** Listagem dinâmica de produtos.
- **Produto:** Detalhes específicos de cada item.
- **Carrinho:** Gerenciamento de itens selecionados.
- **Checkout:** Fluxo de finalização de compra.

## Tecnologias
| Tecnologia | Uso |
| --- | --- |
| HTML5 | Estruturação semântica das páginas |
| Bootstrap 5 | Estilização e componentes responsivos |
| JavaScript | Manipulação de DOM e lógica do carrinho |
| JSON | Armazenamento local dos dados de produtos |[1]
> [!TIP]
> O projeto utiliza o arquivo `products.json` para centralizar as informações dos produtos, facilitando a manutenção e futuras integrações com APIs.
## Como Visualizar
Como o projeto é estático, basta abrir o arquivo `index.html` em qualquer navegador ou utilizar a extensão *Live Server* no VS Code.[1]
